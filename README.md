1️⃣ What is the difference between var, let, and const?
    - var is way to declare variables where it is Function scoped and can be re declared and changed. let is another way to declare variables where it is block scoped and value can be changed but cannot redeclare in the same scope. const is also block-scoped but the value cannot be changed  after assigning. 

2️⃣ What is the spread operator (...)? 
    - The spread operator is used to copy elements from arrays or objects. It is also used to expand.

3️⃣ What is the difference between map(), filter(), and forEach()? 
    - map() runs a function on each element and returns a new array. filter() returns a new array with elements that pass a condition. forEach() just loops through the array, does not return a new array.

4️⃣ What is an arrow function? 
    - When we use arrow like symbol (=>) to write the function in a shorter way then the function is called arrow function.

5️⃣ What are template literals?
    - Those string which are written with backticks (`) are template literals. It allows variables and multiline text easily.
